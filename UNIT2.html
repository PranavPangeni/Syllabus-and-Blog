INTRODUCTION TO NUMBER SYSTEM

The number system is a writing system for expressing numbers; it is a mathematical notation for representing numbers of a given set, using digits or other symbols in a consistent manner. The most common number systems are decimal, binary, octal, and hexadecimal.

Types of Number Systems

  • Decimal Number System: The decimal number system is base 10, using digits from 0 to 9. It is the most widely used number system in the world.
  • Binary Number System: The binary number system is base 2, using only two digits: 0 and 1. It is the foundation of all binary code used in computer programming and digital electronics.
  • Octal Number System: The octal number system is base 8, using digits from 0 to 7. It is used in some computing applications as a shorthand for binary.
  • Hexadecimal Number System: The hexadecimal number system is base 16, using digits from 0 to 9 and letters A to F. It is commonly used in programming and computer science.

1's Complement

The 1's complement of a binary number is obtained by flipping all the bits, changing 0s to 1s and 1s to 0s. To apply 1's complement, follow these steps:

  • Take a binary number, e.g., 1010.
  • Flip the bits: 0101 is the 1's complement.

To subtract two binary numbers using 1's complement:

  • Find the 1's complement of the number to be subtracted.
  • Add the first number and the 1's complement of the second number.
  • If there is a carry, add it to the result.

For example, to subtract 1100 from 1010:

  • 1's complement of 1100 is 0011.
  • Add: 1010 + 0011 = 1101.
  • Since there is no carry, the result is 1101, which is -4 in decimal.

2's Complement

The 2's complement of a binary number is obtained by inverting the bits and adding 1 to the least significant bit. To apply 2's complement, follow these steps:

  • Take a binary number, e.g., 1010.
  • Flip the bits: 0101.
  • Add 1: 0101 + 0001 = 0110 is the 2's complement.

To subtract two binary numbers using 2's complement:

  • Find the 2's complement of the number to be subtracted.
  • Add the first number and the 2's complement of the second number.
  • If there is a carry, it is discarded.

For example, to subtract 1100 from 1010:

  • 2's complement of 1100 is 00100 (invert and add 1).
  • Add: 1010 + 00100 = 1110.
  • The result is 1110, which is -4 in decimal.

INTRODUCTION TO BOOLEAN LOGIC

Boolean logic is a form of algebra in which all values are either true or false. It is fundamental to computer science and digital circuit design.

Boolean Gates

  • AND Gate: Outputs true only if both inputs are true.
  • OR Gate: Outputs true if at least one input is true.
  • NOT Gate: Outputs the opposite value of the input.
  • NAND Gate: Outputs false only if both inputs are true (NOT AND).
  • NOR Gate: Outputs true only if both inputs are false (NOT OR).
  • XOR Gate: Outputs true if the inputs are different.
  • XNOR Gate: Outputs true if the inputs are the same.

De Morgan's Theorems

De Morgan's Theorems are two fundamental rules in boolean algebra that relate conjunctions and disjunctions of boolean variables:

  • First Theorem: The negation of a conjunction is the disjunction of the negations:
    ¬(A ∧ B) = ¬A ∨ ¬B
  • Second Theorem: The negation of a disjunction is the conjunction of the negations:
    ¬(A ∨ B) = ¬A ∧ ¬B

These theorems are essential for simplifying boolean expressions and designing digital circuits.

Q&A Section

1. What is a number system?

A number system is a way of representing numbers using a specific set of symbols and rules. The main types are:

  • Decimal (Base 10)
  • Binary (Base 2)
  • Octal (Base 8)
  • Hexadecimal (Base 16)

2. How does the binary number system work?

The binary number system uses only two digits: 0 and 1. It is used in computers because:

  • Each bit represents an ON (1) or OFF (0) state in digital circuits.
  • It simplifies computation and storage in hardware.
  • All data in computers, including text, images, and sounds, are stored as binary.

3. What is Boolean logic?

Boolean logic is a form of algebra that uses true (1) and false (0) values. It is the foundation of digital circuits and computing. Basic Boolean operations include:

  • AND: Returns true if both inputs are true.
  • OR: Returns true if at least one input is true.
  • NOT: Inverts the input value.

4. What are logic gates?

Logic gates are electronic circuits that perform Boolean operations. The basic gates are:

  • AND Gate: Outputs 1 only if both inputs are 1.
  • OR Gate: Outputs 1 if at least one input is 1.
  • NOT Gate: Outputs the inverse of the input.
  • NAND, NOR, XOR, and XNOR Gates: Variations of AND, OR, and NOT operations.

5. How do you convert a decimal number to binary?

The conversion follows these steps:

  1. Divide the decimal number by 2.
  2. Record the remainder (0 or 1).
  3. Repeat until the quotient is 0.
  4. Read the remainders in reverse order.
Example: Convert 13 to binary: 
                13 / 2 = 6, remainder = 1
                6 / 2 = 3, remainder = 0
                3 / 2 = 1, remainder = 1
                1 / 2 = 0, remainder = 1

Binary equivalent of 13 = 1101

6. What is the significance of hexadecimal numbers in computing?

Hexadecimal (Base 16) is widely used in computing because:

  • It is a compact way to represent binary numbers.
  • Each hex digit corresponds to 4 binary digits (bits).
  • Memory addresses and color codes in web design use hex notation.
Example: Binary 11001100 = Hexadecimal CC.
